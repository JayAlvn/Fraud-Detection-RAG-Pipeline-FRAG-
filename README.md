# Fraud-Detection-RAG-Pipeline-FRAG-
A Python-based Retrieval-Augmented Generation (RAG) pipeline for enterprise 
document verification and fraud detection.

## Overview
This project investigates the limitations of standard RAG when applied to 
enterprise fraud detection document corpora — where relevant signals are spread 
across multiple files, embedded in cross-references, or deliberately obscured 
through vague language. The pipeline combines hybrid retrieval, linguistic 
anomaly detection, and self-reflective hallucination mitigation to surface 
confidence-scored, source-cited alerts for human review.
